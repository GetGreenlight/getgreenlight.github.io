How It Works

1

Connect your agents

Add a hook to your AI coding agents that forwards permission requests to Greenlight's relay server. Supports Claude Code, Windsurf, and more.

2

Get notified instantly

When your agent wants to run a command, edit a file, or take any action, you get a push notification on your iPhone.

3

Approve or deny

Review the action details and tap to allow, deny, or create an "always allow" rule. Your agent continues instantly.

Features

Multi-Agent

One app for all your agents. See which tool sent each request with agent badges. Rules work across Claude Code, Windsurf, and more.

Smart Rules

Create "always allow" rules with auto-generated patterns. Destructive commands use exact matching for safety.

Command Risk Intelligence

Commands are color-coded by risk level using a registry of 170+ CLI tools. See at a glance which parts of a compound command need attention.

Rich Previews

See syntax-highlighted commands, file diffs, and tool-specific views before you approve.

Multi-Project

Project badges show which codebase a request is coming from. Rules are scoped per-project.

Emergency Stop

Deny and halt the agent entirely with a single tap. Regain control immediately.

Interactive Questions

Respond to agent questions with radio buttons, checkboxes, and free-text answers directly in the app.

Live Activity Stream

Watch Claude Code work in real time. See every tool call, file read, and command as it happens — right from your phone. Greenlight Pro.

Push Notifications

Get alerted even when the app is closed. Time Sensitive alerts break through Focus modes so you never miss a request. Greenlight Pro.
